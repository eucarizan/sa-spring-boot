# Module 8 - Actuator
## Agenda
In this lab you will gain experience with Spring Boot Actuator and its features

## Lab
###  Learning Outcomes
You will learn how to:
1. Configure Spring Boot Actuator
2. Expose some or all Actuator endpoints
3. Define custom metrics
4. Extend the */actuator/healt* endpoint to add custom health checks

You will be using the *44-actuator* project.

Estimated time to complete: 45 minutes.
